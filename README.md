# study-group-finder-platform
this repo is for our web development competition
Project Description

This allows students to:

- Sign up & Login securely  
- View and edit their profile  
- Enroll in courses  
- View recommended subjects if not enrolled  
- Create student study groups  
- Search for subjects, groups, and other modules  
- View announcements and messages  
- See detailed module content when clicked from the left-side menu  

This system makes academic communication and course management easier for students.
